## Реализация клиент-серверного взаимодействия через UDP-протокол с использованием `UdpClient`.

### 🚀 Запуск проекта
Запустите сервер:

```bash
cd Server && dotnet run
```
Запустите клиент (в другом терминале):
```bash
cd Client && dotnet run
```
### 💻 Основной функционал

`UDP-сервер (Server/Program.cs)`
- Ожидание подключений
- Обработка входящих сообщений
- Ответ клиенту
- Использование CancellationToken, чтобы можно было правильно останавливать работу сервера

`TCP-клиент (Client/Program.cs)`
- Подключение к серверу
- Отправка сообщений
- Получение ответа
- Ввод ключевого слова "Exit" для завершения программы
